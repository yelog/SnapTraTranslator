# SnapTra Translator

[English](README.md) | [简体中文](README.zh-CN.md) | [한국어](README.ko.md)

カーソル下の単語をスクリーンキャプチャと OCR で即時翻訳する、軽量な macOS メニューバーアプリ。ホットキーを押してテキスト上にカーソルを合わせると、翻訳、発音記号、辞書定義を表示する美しいフローティングバブルが現れます。

## 機能

### コア翻訳
- **即時 OCR 翻訳** - カーソル周围的スクリーン領域をキャプチャし、ポインターに最も近い単語を検出
- **フローティングバブル** - 翻訳結果とともにカーソル近くに表示されるモダンな半透明バブル
- **辞書定義** - 品詞（名詞、動詞、形容詞など）ごとにグループ化された詳細な単語定義
- **発音記号** - 認識された単語の発音記号を表示
- **テキスト読み上げ** - 翻訳後にオプションで発音を再生

### 翻訳モード
- **連続翻訳** - ホットキーを押しながらマウスを動かすと翻訳を継続
- **シングルルックアップモード** - ホットキーを 1 回押すごとに 1 回翻訳、バブルはインタラクティブ（コピー、閉じるボタン）

### サポート言語
簡体字中国語、繁体字中国語、英語、日本語、韓国語、フランス語、ドイツ語、スペイン語、イタリア語、ポルトガル語、ロシア語、アラビア語、タイ語、ベトナム語

### カスタマイズ
- **単一キーホットキー** - 修飾キー（Shift、Control、Option、Command、Fn）からトリガーキーを設定
- **言語選択** - 翻訳言語ペアを選択
- **ログイン時に起動** - ログイン時に自動起動
- **OCR デバッグ領域** - トラブルシューティングのためにキャプチャ領域と検出された単語の境界ボックスを可視化

### その他の機能
- **クリップボードにコピー** - 単語や翻訳をクイックコピー
- **学習単語のエクスポート** - 調べた単語を TXT、Anki TSV、CSV として保存
- **言語パック検出** - 不足している言語パックを自動的に確認して通知
- **メニューバーアプリ** - Dock を散らかさずにメニューバーで静かに動作

## 要件

- macOS 14+（翻訳機能には macOS 15 のシステム翻訳 API が必要）
- 画面収録权限（OCR キャプチャに必須）

## ビルドと実行

Xcode で開く：
```bash
open "SnapTra Translator.xcodeproj"
```

コマンドラインでビルド：
```bash
# デバッグビルド
xcodebuild -project "SnapTra Translator.xcodeproj" -scheme "SnapTra Translator" -configuration Debug build

# リリースビルド
xcodebuild -project "SnapTra Translator.xcodeproj" -scheme "SnapTra Translator" -configuration Release build

# クリーン
xcodebuild -project "SnapTra Translator.xcodeproj" -scheme "SnapTra Translator" clean
```

## 使用方法

1. **権限を付与** - アプリを起動し、画面収録权限を付与
2. **設定を構成** - 設定ウィンドウで好みのホットキーと言語ペアを設定
3. **翻訳** - ホットキーを押しながらテキスト上にカーソルを合わせると、翻訳、発音記号、定義とともにバブルが表示
4. **閉じる** - ホットキーを離して閉じる（シングルルックアップモードでは X をクリック）

## トラブルシューティング

- **バブルが表示されない** - システム設定 > プライバシーとセキュリティ > 画面収録で権限を確認
- **macOS 15 で翻訳が欠落** - システム設定 > 一般 > 言語と地域 > 翻訳言語で言語パックをインストール
- **ホットキーが動作しない** - 他のアプリが同じキーを使用していないか確認し、別の修飾キーを試す
- **バブルが画面端で切り取られる** - 「OCR デバッグ領域」を有効にしてキャプチャ領域を確認し、カーソル位置を調整
