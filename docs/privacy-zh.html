隐私政策

SnapTra Translator

最后更新：2026 年 1 月 28 日

隐私优先：SnapTra Translator 在设计时充分考虑了您的隐私。我们不收集、存储或传输任何个人信息。所有翻译都在您的设备上本地完成。

概述

SnapTra Translator（"本应用"）是一款 macOS 应用程序，使用光学字符识别 (OCR) 和 Apple 内置的翻译框架提供屏幕文字的即时翻译。本隐私政策说明应用如何处理信息。

我们不收集的信息

本应用不收集、传输或共享以下任何信息：

  • 个人信息：不收集姓名、电子邮件地址、电话号码或其他个人身份信息
  • 屏幕内容：通过 OCR 捕获的文字在本地处理，绝不会发送到我们的服务器或任何第三方
  • 翻译历史：应用不存储或追踪您的翻译历史
  • 使用统计：我们不收集分析数据、崩溃报告或使用统计信息
  • 位置数据：应用不访问或收集您的位置信息
  • 设备信息：不收集设备标识符、系统信息或硬件详细信息

本地存储在您设备上的数据

应用仅出于功能目的在您的 Mac 上本地存储以下信息：

用户偏好设置

  • 快捷键设置：您选择的用于激活翻译的键盘快捷键
  • 语言偏好：选择的源语言和目标语言
  • 功能设置：如"播放发音"、"持续翻译"和"登录时启动"等偏好设置

系统权限

应用请求以下 macOS 权限：

屏幕录制权限

  • 用途：从屏幕捕获文字以进行 OCR 翻译
  • 使用方式：仅在您按下快捷键时捕获光标周围的小区域
  • 存储：捕获的图像在内存中处理，翻译后立即丢弃
  • 不录制：应用不录制或保存您的屏幕。它仅对捕获的文字执行实时 OCR

注意：您可以随时在系统设置 > 隐私与安全性 > 屏幕录制中撤销此权限。

屏幕录制数据说明

本节根据 App Store 审核要求，说明应用对屏幕录制数据的使用方式。

使用屏幕录制的功能

  • 即时 OCR 翻译：当您按下快捷键时，应用会捕获光标周围的小区域以识别屏幕文字。
  • 持续翻译（可选）：启用后，应用会在您移动鼠标时周期性捕获光标附近的小区域，以提供连续翻译。

通过屏幕录制收集的数据

  • 屏幕图像：光标周围的小范围图像，可能包含屏幕文字和附近的界面元素。
  • 识别文本：从图像中通过 OCR 提取出的文字。

收集与使用目的

  • 用于 OCR 识别屏幕文字。
  • 用于翻译识别到的文字并在悬浮气泡中显示结果。

共享与披露

  • 不与第三方共享：屏幕录制数据不会分享给任何第三方。
  • 不上传服务器：所有处理均在本地设备完成。

存储与保留

  • 仅在内存中处理：捕获的图像和 OCR 文本仅在内存中处理。
  • 即时丢弃：翻译完成后立即丢弃相关数据。
  • 无持久化存储：不会把屏幕录制数据保存到磁盘或同步到云端。

翻译工作原理

所有翻译处理都在您的设备上本地完成：

  1. 当您按下快捷键并将光标悬停在文字上时，应用捕获光标周围的小区域
  2. Apple 的 Vision 框架（macOS 内置）执行 OCR 从图像中提取文字
  3. Apple 的翻译框架（macOS 15+ 内置）翻译文字
  4. 翻译后的文字显示在悬浮气泡中
  5. 所有捕获的图像和文字在显示后立即丢弃

不会向外部服务器发送数据。所有处理都使用 Apple 内置的设备端框架。

第三方服务

应用不使用任何第三方服务、分析工具或广告网络。

数据安全

  • 应用存储的所有数据都保存在您的设备本地
  • 应用使用 macOS 沙箱以提供额外的安全性
  • 不会为翻译或数据收集目的建立网络连接
  • 您的数据受 Mac 内置安全功能的保护

儿童隐私

应用不会有意收集儿童的信息。由于我们不收集任何个人信息，本应用对所有年龄段的用户都是安全的。

您的权利

由于应用不在我们的服务器上收集或存储个人信息：

  • 数据访问：您的所有数据都本地存储在设备上，仅您可以访问
  • 数据删除：您可以通过卸载应用或在应用设置中重置偏好设置来删除所有应用数据
  • 数据可移植性：不适用，因为我们不收集数据

隐私政策的变更

我们可能会不时更新本隐私政策。任何更改都将在本页面上发布，并更新"最后更新"日期。继续使用应用即表示接受更新后的政策。
