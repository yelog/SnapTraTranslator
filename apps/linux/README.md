# Linux Native Shell Placeholder

This directory is reserved for the future native Linux shell.

Planned scope:
- GTK4 + libadwaita
- GNOME / Wayland first
- XDG Desktop Portal for capture and permission-sensitive integration
- Shared dictionary assets and text-processing contracts from the main repository

Rules for future work:
- Do not add unfinished build systems until the macOS-side service boundaries are proven stable.
- Keep Linux shell code separate from the macOS target.
- Prefer native desktop APIs over generic cross-platform UI frameworks for the first release.
